# Calculadora CLI

Calculadora de línea de comandos que realiza operaciones básicas entre dos números

## Ejecución

'''bash
python main.py

### Operaciones

1. Suma
2. Resta
3. Multiplicación
4. División con validación de división por cero

#### Conceptos aplicados

- Tipos de datos: int
- Conversión de tipos con int()
- Operadores aritméticos: +, -, *, /
- Condicionales if/elif/else
- Operadores de comparación !=
- Manejo básico de errores
